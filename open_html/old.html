Open HTML is a small Website to break the Safari "Sandbox" and open html files with css and js on IOS or IPadOS.

Feature Infos are under /open_html/v1/feature.txt or v2.

The version below is the simplest Type of this Website. The only function of this Website is to open HTMLs.

Version 1.1.0

Version 2 is a bit more complicated. In version 2 I have added Windows and Settings, but it still consists out of one file. (Doesn't support Touch Devices yet!)

Version 2.1.0

Version 3-a (alpha) is a version for which you need to host a server (currently nodejs) and it consists of multiple files and folders. It adds (not yet) support for Filesystem and Terminal accsess.
